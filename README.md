# User Management App (MERN)

## Setup

1. **Backend**
   ```bash
   cd backend
   npm install
   npm run dev
   ```
   Server runs on http://localhost:5000

2. **Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   Client runs on http://localhost:5173

## Environment Variables

Check `backend/.env` (created automatically).
Default Admin:
- Email: admin@example.com
- Password: admin123

## User Rules
- Register/Login via UI.
- Profile image upload supported.

## Admin Rules
- Go to `/admin` to login.
- Manage users in Dashboard.
